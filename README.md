# C-FOX - AI CFO Copilot

AI-powered financial visibility and decision support platform for startups and small businesses.

## Features

- Monthly revenue and expense tracking
- Key financial metrics calculation (burn rate, runway, cash flow)
- AI-powered forecasting
- Natural language financial Q&A
- What-if scenario analysis
- Demo dataset for quick testing

## Installation
```bash
npm install
npm run dev
```

## Usage

1. Upload monthly financial data (CSV) or use demo dataset
2. Enter business context (stage, team size, targets)
3. View calculated metrics and forecasts
4. Ask questions in natural language
5. Run scenario analyses for growth decisions

## Technology Stack

- React 18
- Recharts for visualization
- Claude AI for analysis
- Vite for build tooling

## Note

This is an advisory tool focused on visibility and forecasting. Not for accounting or compliance purposes.
